# 완전판매 판정기준 — 근거 자료집

기존 완전판매 실무바이블(2차 해설서) 기반 데이터를 **1차 법령 + 당국 기준 + 실제 판정사례**로 보강하기 위한 공개 출처 조사 자료입니다.

- **조사일**: 2026-07-13
- **조사 방식**: 공개 출처(국가법령정보센터·금융위·금감원·금융투자협회·법원·언론) 웹 조사
- **활용 대상**: 앱의 `근거 조항` 데이터(`heading` / `text` / `document_type` / `document_id`) 및 감점룰 `basis` 연결

## 각 항목 필드 규약

- **heading** — 근거 조항 제목 (앱 스키마)
- **text** — 조항 원문 요지/핵심 내용 (앱 스키마)
- **document_type** — 법률 / 시행령 / 감독규정 / 가이드라인 / 해설서 / 자율규제 / 분쟁조정례 / 제재사례 / 판례 (앱 스키마)
- **document_id** — 출처 문서명 (앱 스키마)
- **판매원칙** — 6대 원칙 분류(적합성·적정성·설명의무·불공정영업·부당권유·광고규제) 또는 절차 (감점룰 매핑용 메타)
- **판정포인트** — 감점/위반으로 본 핵심 행위 한 줄 (감점룰 후보, C층위)
- **출처 URL** — 확인한 공개 출처 링크

## ⚠️ 정리·적재 전 반드시 확인할 사항

1. **자본시장법 투자권유 조항(구 제46·46조의2·47조)은 2021 금소법 제정으로 삭제**되어 금소법으로 일원화됨. 근거 인용은 **자본시장법이 아니라 금소법 제17·19조**로 해야 함. (기존 seed에 자본시장법 인용이 있으면 교체 필요)
2. **금융소비자보호 감독규정(금융위 고시) 원문**(적합성·적정성 평가 산정방식, 위험등급 산정, 부당권유 세부행위, 핵심설명서 기재사항)은 law.go.kr 행정규칙이 JS 렌더링이라 **조문 원문 미확보 → 원문 대조 필요**.
3. 분쟁조정 **조정번호(사건번호)**, 홍콩 ELS 은행별 가감표, 일부 대법원 판결 선고일·사건번호는 언론 기준이라 **결정서 원문 대조 권장**. 각 파일에 "확인필요"로 표기.
